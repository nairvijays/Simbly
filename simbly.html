Repeating this everytime Sounds stupid...

Use Simbly to create namespaced objects

Create a namespaced object

Simbly.Create("com.widgets.name"); Simbly.Create(["com.widgets.name","com.widgets.name1","com.widgets.name2"]);

Create a namespaced module

Simbly.CreateModule("com.widgets.name", function(){}); Simbly.CreateModule("com.widgets.name", {}); Simbly.CreateModule("com.widgets.name", "hello world"); Simbly.CreateModule("com.widgets.name", true);

Verify a namespaced object

var check = Simbly.Verify("com.widgets.name");
